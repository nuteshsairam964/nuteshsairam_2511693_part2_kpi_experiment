# nuteshsairam_2511693_part2_kpi_experiment

## Business Problem

The company launched a new onboarding campaign to improve user activation and conversion. The objective is to determine whether the treatment experience performs significantly better than the existing onboarding process.

The decision is whether the treatment should be rolled out to all users. Success will be evaluated primarily using Paid Conversion Rate while monitoring guardrail metrics such as refund rate, support ticket rate, engagement score and average days to convert.
